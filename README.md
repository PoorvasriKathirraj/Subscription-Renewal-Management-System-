Subscription Renewal Management System is a Java-based web application developed using Servlets and JDBC.
The system allows users to add, view, and manage subscription renewal records efficiently.
It uses Oracle Database for storing renewal details securely.
The application follows a layered architecture including Bean, DAO, Service, and Servlet packages.
Users can generate unique Renewal IDs automatically using database sequences.
It supports viewing individual renewal records as well as all renewal entries.
The project demonstrates CRUD operations, database connectivity, and MVC design principles.
This system is suitable for managing subscription renewals in small to medium-scale organizations.
